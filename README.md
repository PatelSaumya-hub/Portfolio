# Student Portfolio

A clean, minimalist, plain-white React + Vite portfolio application built as part of the student portfolio lab assignment.

## Reusable Components
- **NavBar**: Highlights the active scrolled section.
- **Header**: Accepts name and themeColor inline styling.
- **About**: Displays software engineering background and selected internship/hackathon highlights.
- **Skills**: Accepts a skillList array prop and renders elements dynamically.
- **Footer**: Minimal contact details.

## Engineering Projects
- **Paper Trail**: Academic NLP similarity matching tool.
- **ChainBeasts & PowerToken**: Solidity game evolution contract system.
- **Walmart Sales Analytics**: Jupyter sales correlation data study.

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the Vite local development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.
